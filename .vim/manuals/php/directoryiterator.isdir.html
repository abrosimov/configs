(PHP 5 <= 5.1.1)

DirectoryIterator::isDir — Returns true if file is directory

Description

bool DirectoryIterator::isDir ( void )
Warning

Parameters

This function has no parameters.

Return Values

TRUE if it is a directory, otherwise FALSE
